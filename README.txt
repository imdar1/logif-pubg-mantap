# logif-pubg-mantap

Cara Menjalankan Program.

Program ini dibuat untuk bisa di compile dengan GNU Prolog.
Cara meng-compile:
1. compile program dengan instruksi: gplc cobardy.pl
2. Setelah masuk ke prompt gprolog, consult dengan instruksi: [cobardy].
3. Masukkan instruksi : start untuk memulai permainan

*Note: semua instruksi harus diakhiri dengan titik dan enter.
